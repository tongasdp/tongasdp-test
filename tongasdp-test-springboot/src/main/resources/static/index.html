漏洞测试靶场 v1.0
本页面为 RASP 技术测试整合页。包括自写测试样例及部分网上漏洞复现或测试文件。
用以汇总和测试 RASP 技术产品针对不同种类漏洞的 Hook 点的效果及尝试进行绕过。
并且使用本产品默认配置及百度开源产品 [OpenRASP-v1.3.2] official 全部功能配置 block 进行测试。
并对每个测试结果进行了标注，左侧为本产品的测试结果，右侧为 OpenRASP 的测试结果，其中绿色为成功防御攻击，红色为不能防御攻击，黄色为经测试该 Hook 项可能对正常功能造成了影响。
